Pacman Engine Workings

  • Game Start – XNA Game Constructor
    • Create Stage Object – Stage Constructor
      • Creates _loadScenes, _activeScenes, _scenesToUpdate, Stage.Instance
      • Adds Stage as Game component (so that next game functions also run stage functions)
    • Stage – Load Scene(s)
      • Create MenuScene and add to _activeScenes and _loadedScenes
      • Create AnimationTest and LevelScene and add to _loadedScenes
  • Game Initialize – Stage Initialize
    • Specify game content and Stage.IsInitialized = true
    • Runs LoadContent
  • Game Load Content – Stage LoadContent
    • Specify game spriteBatch
    • Loop through each _loadedScene and initializes scene
    • Scene loops through each child and Initializes and LoadContent
  • Game Update – Stage Update
    • Update input helper
    • Create a temp array _scenesToUpdate from _activeScenes
    • If enabled, update active scenes
      • Scene Update – If child.enabled, update child
  • Game Draw – Stage Draw
    • Loop all _activeScenes and if scene.visible, draw scene

Menu Scene Description

  • Menu Scene Constructor
    • Create ImageObject pacman [x: 125, y: 125, scale:1.1]
    • Create SpriteObject pacman2 [index: 5, x: 300, y: 0]
    • Create AnimatedSpriteObject pacman3 [seq: 36|37|36|38, time: 150ms, x: 350, y: 0]
    • Create RectangleObject [width: 32, height: 32, x: 300, y: 50]
    • Create CircleObject [radius: 16, x: 350, y: 50]
    • Create TextObject [text: "Woot!", x: 300, y: 100, size: 1.5, rotation: -15, color: lime]
    • Create TextObject [text: "Alot of text about spacebar", x: 325, y: 350, size: 2, rotation: -5]
    • All objects added to scene
  • Menu Scene Update
    • If {Space} is pressed, goto Level Scene
    • If {T} is pressed, goto AnimationTest Scene
    • If {Esc} is pressed, Exit Game
